## Terminal File Explorer

<img width="1686" height="990" alt="image" src="https://github.com/user-attachments/assets/048ff34f-9936-45ff-a157-36d14552ec3e" />
Visual file system explorer in the terminal, use the mouse or keyboard, supports mouse on linux without any display server, or WM, Macos, Linux. (maybe windows some day)

## Written with Prism C
This project is a demo project of Prism, the [Robust C transpiler](https://github.com/dawnlarsson/prism)

## Build & Run
```sh
prism run explore.c
```

or just build:
```sh
prism -O3 -s -flto explore.c -o explore
```
